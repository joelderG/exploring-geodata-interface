Gesture Explanation
Second Touch
Activates the context menu
Swipe Left-Right
Cycles the cutting plane orientation (XZ/YZ)
Swipe Right-Left
Cycles the cutting plane orientation in reverse
Swipe Top-Bottom
Resets the cutting plane orientation to XY
Second Touch Drag Left
Toggle visibility of the selected class
Second Touch Drag Right
Toggle visibility of all other classes
Second Touch Drag Down
Reset all class visibility changes
